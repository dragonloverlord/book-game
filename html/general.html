General Help


  1. How to play

    All you have to do is click on the book you want to write and then wait. You may also buy and even unlock some additional book types.

  2. How to save

    In this game you must click the save button in order to save as this game does not auto save.

  3. What type of game is this

    This is a idle game.

  4. What is a idle game

    An idle game is a game that is designed to be long lasting and relatively easy going.
